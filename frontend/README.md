# Эколайф — фронтенд

Сайт компании по утилизации отходов. Чистый HTML, обычный CSS (БЭМ), Tailwind для утилит, форма через fetch. Без сборки и без node_modules.

## Структура проекта

```
new-frontend/
├── index.html          # Главная страница
├── css/
│   └── main.css        # Все стили (БЭМ + анимации), обычный CSS
├── js/
│   └── main.js         # Меню, скролл-анимации, отправка формы (fetch)
├── assets/
│   └── images/         # Логотип: скопируйте сюда 222.png как logo.png
└── package.json
```

## Запуск

Откройте `index.html` в браузере или поднимите статический сервер: `npx serve .`

## Форма обратной связи

- Отправка через **fetch** на ваш API.
- У формы задайте атрибут `data-action="/api/feedback"` (или ваш URL).
- Отправляются поля: `name`, `phone`.

## API для бэкенда

Фронтенд отправляет запрос на один эндпоинт. Бэкенд должен его реализовать.

### `POST /api/feedback` — отправка заявки на обратную связь

Принимает заявку из формы обратной связи (имя и телефон).

**Request:**

- **Method:** `POST`
- **Content-Type:** `application/json`
- **Body (JSON):**
  ```json
  {
    "name": "Имя клиента",
    "phone": "+7 999 123-45-67"
  }
  ```

**Поля:**

| Поле    | Тип    | Обязательно | Описание                          |
| ------- | ------ | ----------- | --------------------------------- |
| `name`  | string | да          | Имя клиента (валидация на фронте) |
| `phone` | string | да          | Телефон (валидация на фронте)     |

**Успешный ответ (2xx):**

- **Status:** `200`–`299`
- **Body:** любой (не используется на фронте)

**Ответ об ошибке:**

- **Status:** `4xx` или `5xx`
- **Content-Type:** `application/json`
- **Body:**
  ```json
  {
    "message": "Текст ошибки для пользователя"
  }
  ```
  Поле `message` показывается пользователю вместо стандартного сообщения.

**Особенности:**

- Таймаут запроса на фронте: **10 секунд**
- Пустые `name` или `phone` не отправляются (валидация до запроса)

## Логотип

Если в `assets/images/` нет `logo.png`, используется fallback из папки с исходным проектом. Чтобы везде был один логотип, скопируйте `222.png` в `assets/images/logo.png`.

## Технологии

- **Стили**: обычный CSS (БЭМ), Tailwind CDN для утилит
- **Форма**: нативный fetch
- **Анимации**: появление при скролле (Intersection Observer), переходы в CSS
